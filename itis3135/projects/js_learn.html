What is JavaScript?

If HTML is the skeleton and CSS is the clothes, JavaScript is the brain. It lets websites respond to you — clicking buttons, filling forms, playing videos.

🎮

Almost every game you play in a browser? JavaScript. The search suggestions on Google? JavaScript. Animated menus? JavaScript. It's literally everywhere!

JavaScript is a programming language — it lets you give instructions to a computer. The browser reads those instructions and does what you say!

Variables: Storing Information

A variable is like a labeled box where you store information. You name the box, put something in it, and use it later.

variables.js 
// Create a variable called "name"
let name = "Alex";

// Create a variable called "age"
let age = 12;

// Print them out!
console.log("Hello, my name is " + name);
console.log("I am " + age + " years old");
💡

let means "I'm creating a variable." The = means "the value is." And console.log() prints something so you can see it!

If Statements: Making Decisions

Computers make decisions using if statements. "IF this is true, do this. Otherwise, do something else."

decisions.js 
let temperature = 75;

if (temperature > 70) {
  console.log("It's warm outside! 🌞");
} else {
  console.log("Bring a jacket! 🧥");
}
📝

The > symbol means "greater than." So temperature > 70 asks: "Is the temperature bigger than 70?" If yes → run the first block. If no → run the else block.

Functions: Reusable Actions

A function is a named set of instructions you can run anytime by calling its name. Like a recipe — write it once, use it forever!

functions.js 
// Define the function
function greet(name) {
  console.log("Hello, " + name + "! 👋");
}

// Call it as many times as you want!
greet("Alex");
greet("Jordan");
greet("Sam");
