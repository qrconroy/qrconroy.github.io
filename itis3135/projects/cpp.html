What is C++?

C++ is a compiled programming language — which means you write code, and then a special program (called a compiler) translates it into instructions your computer understands directly.

🎮

Games like Fortnite, Minecraft, and most console games are built with C++. It's also used in Chrome, Windows, macOS, and basically every high-performance program you can think of!

C++ is a bit harder to learn than JavaScript or Python — but understanding it makes you a much stronger programmer overall. Think of it like learning manual transmission before automatic!

🚀 Super fast 🎮 Used in games 💪 Used in OS 🏆 Powers pro software

Your First C++ Program

Every C++ program starts with a similar structure. Here is the classic "Hello, World!" — the first program every programmer learns:

hello.cpp 
#include <iostream>
using namespace std;

int main() {
  cout << "Hello, World!" << endl;
  return 0;
}
// Program output:
Hello, World!

Breaking it down:

CodeWhat it means
#include <iostream>Import the input/output tools
using namespace std;Use standard shortcuts (so we don't have to type "std::" everywhere)
int main() { }The main function — where your program starts
cout << "text"Print text to the screen
return 0;Tell the OS the program finished successfully

Variables & Data Types

In C++, when you create a variable, you have to say what kind of data it holds. This is called a data type.

variables.cpp 
#include <iostream>
using namespace std;

int main() {
  int age = 14;          // whole numbers
  double height = 5.7;   // decimal numbers
  string name = "Alex"; // text
  bool isStudent = true; // true or false

  cout << name << " is " << age << endl;
  return 0;
}
💡

C++ is a "strongly typed" language — you must declare types upfront. This makes programs faster because the computer knows exactly how much memory to reserve!

Loops: Repeating Actions

A loop lets you repeat code multiple times without copy-pasting it. The most common loop is the for loop.

loops.cpp 
#include <iostream>
using namespace std;

int main() {
  // Count from 1 to 5
  for (int i = 1; i <= 5; i++) {
    cout << "Count: " << i << endl;
  }
  return 0;
}
// Program output:
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
📝

The for loop has 3 parts: start (int i = 1), condition (i <= 5 — keep going while true), and step (i++ — add 1 each time).
