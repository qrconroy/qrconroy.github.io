Learning HTML

HTML is the language that builds every webpage you've ever visited. Let's start from scratch — you'll be amazed how fast it clicks!

1

What IS HTML, anyway?

HTML stands for HyperText Markup Language. Don't worry about those big words! In simple terms, HTML is just a way of labeling text to tell the browser what it is.

🤔

Think of it like this: imagine you're writing a letter and you highlight the title in yellow, underline important words, and circle the main points. HTML does the same thing — it wraps labels called tags around your content!

Every webpage you've ever seen — YouTube, Google, Instagram — is built using HTML at its core.

2

Tags: The Building Blocks

HTML uses tags to wrap content. Tags come in pairs — an opening tag and a closing tag. The closing tag has a / before the name.

example.html 
<h1>This is a big heading</h1>
<p>This is a paragraph of text.</p>
<strong>This text is bold!</strong>
💡

Memory trick: Opening tag = door opening 🚪, Closing tag = door closing 🚪. Always close the door!

Here are the most common tags you'll use as a beginner:

Common HTML Tags:

🔷 <h1> to <h6> — Headings (h1 is biggest)
🔷 <p> — Paragraph
🔷 <strong> — Bold text
🔷 <em> — Italic text
🔷 <a> — Links
🔷 <img> — Images
🔷 <ul> / <li> — Lists

Your First Webpage Structure

Every HTML page follows a basic skeleton structure. Think of it like the frame of a house — before you decorate, you need walls!

mypage.html 
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>My very first webpage.</p>
  </body>
</html>
What each part does:

📄 <!DOCTYPE html> — Tells the browser "this is an HTML file"
🌍 <html> — The root container (everything lives inside)
🧠 <head> — Info about the page (not shown to visitors)
📌 <title> — The text in the browser tab
👁 <body> — Everything visitors actually SEE goes here

Links & Images

Links and images use something called attributes — extra information you add inside the opening tag.

links.html 
<!-- A link -->
<a href="https://google.com">Click me!</a>

<!-- An image -->
<img src="cat.jpg" alt="A cute cat">
📝

Note: <img> is a self-closing tag — it has no closing tag!
