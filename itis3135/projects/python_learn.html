Why Python?

Python is one of the most used programming languages on Earth — and for good reason. It reads almost like plain English, which makes it perfect for beginners.

🐍

Variables & print()

In Python, you create a variable just by giving it a name and a value — no need to say what type it is. Python figures that out automatically!

variables.py 
# Create variables
name = "Alex"
age = 12
height = 5.4
is_student = True

# Print them out
print("Hello, my name is", name)
print("I am", age, "years old")

# f-strings: an easier way to combine text
print(f"Hi, I'm {name} and I'm {age}!")
💡

f-strings are Python's superpower for combining text and variables. Put an f before the quotes, then wrap any variable in { }. Way easier than adding strings together!

If Statements & Indentation

Python uses if statements to make decisions. One big difference from other languages: Python uses indentation (spaces) instead of curly braces { } to group code.

decisions.py 
temp = 75

if temp > 80:
    print("It's really hot!")
elif temp > 60:
    print("Nice and warm!")
else:
    print("Grab a jacket!")
📝

Python uses elif (short for "else if") to check multiple conditions. Notice each code block is indented with 4 spaces — this is how Python knows what belongs inside the if statement.

💡

Don't forget the colon : at the end of if, elif, and else lines. Forgetting it is the #1 beginner mistake!

Loops & Lists

Python has incredibly clean loop syntax. A list stores multiple items, and a for loop lets you do something to each item, one at a time.

loops.py 
# A list of fruits
fruits = ["apple", "banana", "mango"]

# Loop through every item
for fruit in fruits:
    print(f"I love {fruit}!")

# Count with range()
for i in range(1, 6):
    print(f"Step {i} of 5")
✨

range(1, 6) generates the numbers 1, 2, 3, 4, 5 (it stops before 6). This is one of Python's most useful built-in tools!
